## Estructura 
Encontramos una vista principal con las peliculas separadas por cards, un navbar con un carrito con contador y y botones de navegacion con la funcionalidad Link. Al hacer click sobre un producto se abre la vista detalle la cual muestra una imÃ¡gen, su nombre, precio, stock y detalle con un contador con botones para agregar o quitar el producto al carrito de compras luego de seleccionar la cantidad deseada.

Ya en el carrito de compras podemos borrar o agregar productos. El formulario esta Hardcodeado y la orden cargada a nuestra base de datos en FireBase.
